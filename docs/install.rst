Installation
============

.. Contents::

Via Virtualenv
--------------

Better to install `rux` with `virtualenv <http://www.virtualenv.org/>`_ 

.. code-block:: bash

    $ mkdir myblog && cd myblog
    $ virtualenv venv
    $ . venv/bin/activate
    $ pip install rux

System-Wide Installation
------------------------

.. code-block:: bash

    $ sudo pip install rux
